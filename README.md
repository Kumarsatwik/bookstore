## Book Store using MERN Stack

Features

- View Books
- Update Book
- Delete Book
- Authentication and Authorization

Project Structure

- Frontend
  - ├───public
  - └───src
    - ├───assets
    - ├───components
    - ├───pages
    - └───reducer
- Backend
  - ├───config
  - ├───controllers
  - ├───middleware
  - ├───models
  - ├───routes
  - └───utils

# Installation

- Clone the repository
  - Enter Frontend/Backend Folder
    - npm install
    - npm run dev (or) npm start
